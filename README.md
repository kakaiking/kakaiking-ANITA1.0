# Anita IDE - The Ultimate Agentic AI Coding Assistant

Anita is a state-of-the-art Agentic AI Integrated Development Environment (IDE) designed to transform how you build software. Combining a premium, high-performance UI with powerful multi-agent coordination, Anita acts as your elite pair programmer.

## üöÄ Getting Started

### Prerequisites
- **Node.js** (v16.0 or higher recommended)
- **Git** (for version control)
- **OpenRouter API Key**: Required for the high-quality chat and reasoning models.

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/kakaiking/kakaiking-ANITA1.0.git
   cd anita-app
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Launch the application**:
   ```bash
   npm start
   ```

## üõ†Ô∏è How to Use

### 1. Workspace Setup
On the first launch, Anita will prompt you to **Select a Workspace**. Choose the root folder of the project you want to work on. Anita will only operate within this directory for safety.

### 2. Configure API Keys
Click the **Gear Icon (Settings)** in the bottom left.
- Enter your **OpenRouter API Key**. 
- Choose your preferred chat model (e.g., `deepseek/deepseek-chat`).

### 3. The Chat Experience
Use the **Chat Panel** on the right for:
- Asking technical questions.
- Explaining existing code.
- Brainstorming architectures.

### 4. High-Octane "Agent Mode"
When you're ready to build a feature:
1. Click the **Agent Icon** in the sidebar.
2. Describe your goal in the composer (e.g., *"Implement a Firebase authentication flow with login/signup pages"*).
3. Anita will enter **Planning Phase**. Review the generated plan.
4. Click **Start Execution**. Anita will then create files, install dependencies, and run commands while keeping you in the loop for approvals.

### 5. Auto-Repair
If an agent command fails or a file cannot be written, Anita's **Repair Mode** will automatically analyze the error and suggest a corrective path forward.
